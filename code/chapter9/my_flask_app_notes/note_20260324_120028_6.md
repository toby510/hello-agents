# Tooling Constraint: Python interpreter unavailable in sandbox

The sandbox environment does not expose `python` or `python3` binaries (neither `which python` nor `which python3` succeeds). This prevents syntax checking (`py_compile`), linting (`pylint`), and type checking (`mypy`). All static analysis must be deferred or performed externally.
